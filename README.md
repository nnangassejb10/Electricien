## 📋 Description

Ce site est un **portfolio one-page** conçu pour présenter le profil professionnel de Leon La Puissance, électricien expert au Gabon. Il met en valeur :

- Ses **services** : installation électrique, tableaux, éclairage industriel, groupes électrogènes, mise aux normes
- Ses **chantiers** : 70+ réalisations dans les secteurs militaire, sanitaire, public et social
- Son **parcours** : 10 ans de carrière avec timeline et barres de compétences
- Ses **témoignages** clients et partenaires
- Un **formulaire de contact** interactif

| Technologie | Utilisation |

## 🚀 Comment utiliser ce projet

### 1. Cloner le dépôt
```bash
git clone https://github.com/votre-username/leon-portfolio.git
```

### 2. Ouvrir le projet
```bash
cd leon-portfolio
```
Ouvrez `index.html` directement dans votre navigateur — aucun serveur requis.

### 3. Personnaliser le contenu

Dans le fichier `index.html`, recherchez et remplacez :

- `+241 XX XX XX XX` → le vrai numéro de téléphone
- `leon@exemple.com` → le vrai email
- `images/leon.jpg` → la vraie photo de profil
- `images/chantier1.jpg` → les vraies photos de chantiers
- Les dates de la timeline → les vraies dates de carrière

### 4. Publier sur GitHub Pages

```bash
# Initialiser git (si pas encore fait)
git init
git add .
git commit -m "Initial commit - Portfolio Leon La Puissance"
git branch -M main
git remote add origin https://github.com/votre-username/leon-portfolio.git
git push -u origin main
```

Ensuite, dans GitHub → **Settings** → **Pages** → sélectionner la branche `main` → **Save**.
